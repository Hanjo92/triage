# 🧩 Triage (트리아지) MVP

> **"완벽한 루틴보다, 끊기지 않는 끈적한 연결감이 중요합니다."** <br/>
> 자기관리에 지치거나 루틴이 자주 끊기며 압박감을 느끼는 현대인을 위한 **초경량 루틴 미니앱 아키텍처**입니다.

<br/>

## 🚀 About "Triage"
트리아지(Triage)는 하루의 멘탈 상태와 에너지 레벨을 단 10초 만에 파악하고, 당장 실행 가능한 가장 작은 단위의 행동(**One-tap Win**)을 추천해 주는 습관/루틴 관리 서비스입니다.
실패에 대한 좌절감이나 무거운 입력값을 덜어내기 위해 **부분 완료를 인정**해 주는 소프트 스트릭(Soft Streak)과, 연속 이탈 시 질책 대신 가장 쉬운 일(예: 물 한 잔 마시기)을 권하는 **회복 모드(Recovery Mode)**를 최우선으로 설계했습니다.

<br/>

## ✨ Key Features

- **📊 10초 모닝 체크인**: 에너지, 가능 시간, 멘탈 상태를 직관적으로 체크
- **🎯 동적 원탭 윈(One-tap Win) 추천**: 알고리즘과 상황 태그(모드, 에너지, 시간 등)에 맞춰 즉시 수행 가능한 행동 3가지 추천
- **🛡️ 회복 모드 (Recovery)**: 루틴이 끊긴 유저를 부드럽게 복귀시키는 초저강도 방어 미션 제공
- **🤝 소프트 스트릭 (Soft Streak)**: 완벽하지 않아도(0.7점) 시도(0.4점)나 출석(0.2점)을 인정하여 끊기지 않는 성취감 유지
- **⏰ 자체 스케줄링 푸시 알림**: 매일 정해진 시각(오전 8시) 아침 체크인 리마인더 
- **🌐 통합 1-Core 듀얼 플랫폼**: 하나의 백엔드 두뇌로 **텔레그램 봇(Telegram Bot)**과 **웹 뷰 미니앱(WebApp)**을 동시 운영

<br/>

## ⚙️ Getting Started (로컬 환경 실행 안내)

이 프로젝트를 로컬 컴퓨터에서 동작시키고 테스트하려면 아래의 과정을 따라주세요.

**1. 패키지 설치 및 DB 세팅**
```bash
# 필수 의존성 설치
pip install -r requirements.txt

# SQLite 데이터베이스 생성 및 MVP 행동 데이터셋 주입
python scripts/seed_actions.py
```

**2. 텔레그램 봇 실행**
- 봇 토큰 발급 (BotFather) 후, 터미널 환경변수에 등록합니다.
```bash
# Windows
set BOT_TOKEN=본인의_텔레그램_토큰
# Mac/Linux
export BOT_TOKEN="본인의_텔레그램_토큰"

python src/platforms/telegram/bot.py
```

**3. 미니앱 (웹 레이어) 프론트/API 서버 실행**
- 새로운 터미널 창을 열고 아래 명령어로 FastAPI 서버를 띄운 뒤, `http://127.0.0.1:8000` 에 접속합니다.
```bash
python -m uvicorn src.platforms.toss.main:app --reload
```
(*또는 텔레그램 봇 채팅에서 미니앱 켜기 버튼의 URL을 자신의 ngrok 포트포워딩 주소로 교체하여 결합 테스트가 가능합니다.*)

## 🚀 Deployment (배포 가이드)
배포 설정이 완벽히 구성(1-Click 수준)되어 있으므로 **Railway.app** 등 클라우드 컨테이너에 즉시 배포할 수 있습니다.
1. GitHub Repository에 현재 완성된 코드를 푸시합니다.
2. Railway 메뉴에서 `New Project` ➜ `Deploy from GitHub repo`를 클릭합니다.
3. PostgreSQL 플러그인을 추가합니다.
4. Variables 메뉴에서 `BOT_TOKEN`만 명시적으로 입력해 줍니다. (`DATABASE_URL` 등은 자동 적용됨)
5. `start.sh` 스크립트를 통해 자동으로 봇과 미니앱 웹 서버가 백그라운드에서 동시 기동됩니다.
