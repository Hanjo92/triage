# Triage Wireframe

트리아지 화면 설계서 (와이어프레임)
1. 홈/시작 화면
오늘 상태 어때요?[좋음] [보통] [힘듦]→ 클릭 시 다음 단계 이동
2. 상태 입력 화면
가능 시간 선택: [3분] [10분] [30분+]상태 선택: [집중됨] [산만] [지침] [압박]
3. 원탭 윈 화면
추천 행동 3개 표시[이걸로 할게요] [다른거 추천]
4. 실행 리마인드
지금 3분만 할까요?[지금] [나중에]
5. 완료 체크 화면
[완료] [조금함] [못함]
6. 결과 피드백
완료/부분/실패에 따른 메시지 출력
7. 회복 모드 화면
가장 쉬운 행동 3개 제안→ 선택 시 복귀 처리
8. 스트릭 화면
최근 7일 점수 표시 (연속성 기반)
9. 설정 화면
알림 시간, 톤, 난이도 설정
