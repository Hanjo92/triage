# Triage Prd

트리아지 미니앱 기획서 (PRD)
1. 제품 개요
하루를 10초 안에 정리하고, 한 번의 탭으로 실행하며, 실패해도 부드럽게 복귀시키는 초경량 루틴 미니앱
2. 핵심 기능
- 10초 모닝 체크인- 원탭 윈 추천- 소프트 스트릭- 회복 루틴- 간단한 완료 기록
3. 타겟 사용자
- 자기관리가 자주 끊기는 사용자- 생산성 앱에 피로감을 느낀 사용자- 작은 실행부터 다시 시작하고 싶은 사용자
4. 사용자 플로우
아침 체크인 → 원탭 윈 선택 → 실행 → 완료 → 스트릭 반영 → (실패 시) 회복 루틴
5. 플랫폼 전략
- 텔레그램: 개인 루틴 MVP- 디스코드: 커뮤니티 실행- 토스: 생활 습관형 UX
6. 핵심 지표
- 체크인율- 완료율- 복귀율- 회복 성공률
7. MVP 범위
체크인, 원탭 윈, 완료, 스트릭, 회복 기능만 포함
8. 개발 일정 (8주)
1~2주 기획 → 3~4주 개발 → 5~6주 테스트 → 7~8주 확장
9. 제품 원칙
- 입력 최소화- 실패 허용- 작은 실행 중심- 부드러운 복귀 설계
